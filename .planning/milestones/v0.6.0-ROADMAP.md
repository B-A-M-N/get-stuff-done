# Milestone v0.6.0: Open Brain Foundations

**Status:** SHIPPED 2026-03-27
**Phases:** 55-55
**Total Plans:** 4

## Overview

This milestone added Open Brain as a separate local-first semantic-recall sidecar without weakening the existing execution-truth boundary. The shipped work covers schema/bootstrap foundations, local embedding posture, curated ingestion, bounded retrieval, feedback-driven reranking, workflow-context integration, degraded-mode safety, and installed Codex operator-path repair.

## Key Accomplishments

- Added a dedicated `gsd_open_brain` schema, graph-ready relational tables, and an optional readiness surface separate from `gsd_local_brain`.
- Introduced a local embedding adapter that targets `fastembed` by default but degrades truthfully when the capability is unavailable.
- Added curated normalized-artifact promotion, bounded ranked retrieval, and explicit recall-outcome feedback loops.
- Wired a bounded `open_brain_recall` pack into workflow context while preserving Firecrawl as the external-context boundary and Second Brain as execution truth.
- Repaired the installed Codex `brain open-status --raw` path so the shipped operator surface reports bounded degraded truth from real installed entrypoints.
