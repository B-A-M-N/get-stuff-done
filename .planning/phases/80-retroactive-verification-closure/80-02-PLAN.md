## Locked Decisions

Decision: consume the shared retro-verification helper from `80-01` rather than hand-authoring two new verification formats.
Decision: summaries and `TRUTH.yaml` remain claim inventories only; neither may serve as final proof for `TRUTH-DEGRADE-01` or `TRUTH-MEMORY-01`.
Decision: Phase 75 verification must explicitly prove fail-closed and degraded-state behavior from current sanctioned surfaces, not from narrative descriptions of prior work.
Decision: Phase 77 verification must explicitly prove governance narrowing, warn-only warnings, and recovery-route availability without weakening hard truth-transition blocks.
Decision: if the current environment cannot fully re-prove a memory or governance claim, the affected row must remain `CONDITIONAL` or `INVALID`; no fabricated pass state is allowed.
Decision: Phase 80 still does not rewrite summaries, truth artifacts, roadmap traceability, or milestone bookkeeping.

## Risks

Risk: Phase 75 can be overstated if memory fail-closed posture is inferred from degraded-mode route blocking instead of directly evidenced.
Guardrail: cite current memory- and verification-facing commands, tests, and source surfaces separately from general degraded policy output.
Fallback: keep `TRUTH-MEMORY-01` conditional if the evidence only proves adjacent enforcement and not the memory boundary itself.

Risk: Phase 77 can blur policy definition with enforcement outcome and miss the distinction between `warn_only`, `recovery_only`, and hard-gated truth transitions.
Guardrail: verify both the policy map/helper and the CLI enforcement behavior with separate evidence rows.
Fallback: if the policy map exists but enforcement proof is partial, downgrade the affected truths instead of collapsing the distinction.

Risk: late-phase global degraded posture could leak into the verification artifacts as a generic disclaimer rather than a mechanically reasoned downgrade.
Guardrail: tie each downgrade to a concrete blocked command, stale artifact, or unavailable direct evidence source.
Fallback: preserve precise phase-local evidence and note global posture only where it actually changes proof strength.
