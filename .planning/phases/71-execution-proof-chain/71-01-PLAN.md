## Locked Decisions

Decision: Git and runtime are proof authorities; Plane and other coordination layers cannot prove execution.
Decision: Proof is dual-layer: task proof is atomic, and summaries are structured proof indexes.
Decision: The proof artifact model is hybrid, with machine logs as the truth layer and summaries as the representation layer.
Decision: No-code-change tasks are valid only through an explicit evidence exception artifact, not implicit “already done” claims.
